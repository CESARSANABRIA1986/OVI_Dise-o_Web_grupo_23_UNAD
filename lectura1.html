Diseño de Sistios Web

Diseño Web, esta en tus manos.

Para el desarrollo de las Paginas WEB, debemos desarrollar primero que todo un Sketch, el cual es un primer boceto que realizamos para un proyecto digital que queremos crear, posteriormente debemos, desarrollar un Wireframe, el cual permitirá especificar la asignación de espacio y priorización del contenido, las funcionalidades disponibles, y los comportamientos deseados, después de estructurar el boceto y la asignación de espacio de una pagina web, debemos representar de forma mas avanzada el diseño web para ello debemos crear un Mockup, el mockup es un medio de representación de la apariencia del producto.

QUE ES HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas o el uso de imágenes y tablas de datos. Como lo sugiere el título, este artículo te dará una comprensión básica de HTML y cúal es su función. Otra defición es: HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto.
